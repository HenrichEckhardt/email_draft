Genesys Email Bot Draft Mockup

This widget simulates an email bot that prepares two possible answers. The agent selects one answer and an injection mode, then the widget reads the current active draft, merges the selected answer with the draft, discards the original draft, and recreates it with the complete merged content.

Email bot prepared answers

Select one mock answer. The selected content is editable before injection.
In before signature/history mode, the insertion point is detected using common markers such as HTML elements with class/id containing signature, a text signature delimiter --, quoted reply markers, or blockquote/history sections. If no marker is found, the selected answer is appended to the current draft body. The other modes preserve the same discard-and-recreate flow but place the selected answer before, after, or instead of the current draft body.
